# othello

A command-line interface for Othello! Either player can be human or a bot.
A list of available agents is available via the help command.


## Installation

To install the Othello project, run this command in your terminal:

```
$ pip install othello
```


## Usage

```
$ othello [OPTIONS]

    -b <agent>, --black <agent>
    -w <agent>, --white <agent>
    -v, --version
    -h, --help
```
